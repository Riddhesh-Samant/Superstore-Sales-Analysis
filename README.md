# Superstore Sales Analysis

This project focuses on analyzing sales performance, profitability, and customer behavior using Python.

## Objective
To perform exploratory data analysis (EDA) on Superstore sales data and extract meaningful business insights.

## Tools Used
- Python  
- Pandas  
- Matplotlib  

## Key Analysis
- Sales and profit analysis by region  
- Profitability by category and sub-category  
- Identification of loss-making orders  
- Relationship between quantity and profit  

## Key Insights
- Some sub-categories consistently generate losses  
- Quantity has a weak correlation with profit  
- Discounts significantly impact profitability  

## Dataset
Superstore Dataset (Kaggle)
